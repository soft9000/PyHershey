# PyHershey
A fast and easy way to use Hershey's Fonts under Python 3.

NEW: You will enjoy [this video](https://youtube.com/shorts/jZL-P3qeB5M) and [this code](https://github.com/soft9000/PyHershey/blob/master/Hershey9000/DrawOne.py) to get started using these fonts in your own 'spirations.

You can also:

(1) Download the dist/[whatever].whl

(2) ```python -m pip install [whatever].whl```

(3) ```python -m Hershey9000```
